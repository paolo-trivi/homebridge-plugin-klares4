## Ambito

Questi documenti descrivono il comportamento reale del plugin:

- ciclo vita piattaforma Homebridge
- flussi websocket (read/realtime/write)
- routing comandi termostati
- bridge MQTT
- import KSA e cache sanitizzata
- diagnostica operativa
